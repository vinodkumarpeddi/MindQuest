PROJECT OVERVIEW
================================================================================
MindQuest is a comprehensive, production-ready brainstorming platform that
combines real-time collaboration with advanced AI/ML capabilities. It enables
distributed teams to ideate effectively using intelligent features like
automatic clustering, duplicate detection, sentiment analysis, and AI-powered
idea generation.

MACHINE LEARNING TECHNIQUES
================================================================================

1. Text Vectorization:
   - TF-IDF (Term Frequency-Inverse Document Frequency)
   - 384-dimensional feature vectors
   - English stop word filtering

2. Clustering Algorithms:
   - K-Means: Partitional clustering (3-8 clusters)
   - DBSCAN: Density-based clustering
   - Automatic cluster count selection

3. Similarity Measures:
   - Cosine similarity for duplicate detection
   - 85% threshold for flagging duplicates
   - Vector magnitude normalization

4. Natural Language Processing:
   - Sentiment classification (positive/negative/neutral)
   - Keyword extraction (top 5 terms)
   - Automatic tag generation
   - Context-aware text generation

5. Large Language Models:
   - Google Gemini Pro for text generation
   - Cluster label generation
   - Session summarization
   - Idea generation with context

QUICK START
================================================================================

1. Install dependencies:
   npm install
   cd ai_service && pip install -r requirements.txt && cd ..

2. Start all services:
   npm run start:all

   OR use the startup script:
   ./start.sh

3. Access the application:
   Frontend:    http://localhost:5173
   Backend:     http://localhost:3001
   AI Service:  http://localhost:5000

API ENDPOINTS
================================================================================

Authentication:
  POST   /api/auth/register      - Register new user
  POST   /api/auth/login         - Login user
  GET    /api/auth/me            - Get current user

Sessions:
  POST   /api/sessions           - Create session
  GET    /api/sessions           - Get all sessions
  GET    /api/sessions/:id       - Get session by ID
  POST   /api/sessions/:id/join  - Join session
  GET    /api/sessions/:id/export - Export session data

Ideas:
  POST   /api/ideas              - Create idea
  GET    /api/ideas/session/:id  - Get session ideas
  PUT    /api/ideas/:id          - Update idea
  DELETE /api/ideas/:id          - Delete idea
  POST   /api/ideas/:id/vote     - Vote on idea
  POST   /api/ideas/:id/react    - Add reaction

AI Features:
  POST   /api/clusters/trigger        - Trigger clustering
  GET    /api/clusters/session/:id    - Get clusters
  POST   /api/clusters/summarize      - Generate summary
  POST   /api/clusters/generate-ideas - AI idea generation

Python AI Service:
  POST   /check-duplicate        - Duplicate detection
  POST   /cluster                - Cluster ideas
  POST   /summarize              - Generate summary
  POST   /generate-ideas         - Generate new ideas
  GET    /health                 - Health check
